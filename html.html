Pagina web de una tienda de Ropa Deportiva y Accesorios

Un sitio web estático diseñado para personas interesadas en ropa deportiva y accesorios variados. Los usuarios podrán crear una cuenta para acceder al sitio, donde encontrarán diversas categorías como ropa para vóley, fútbol, natación, entre otras. Luego de seleccionar los productos, podrán realizar el pago a través de múltiples métodos de tarjeta.
