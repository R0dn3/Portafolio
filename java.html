El sistema gestiona los votos, permitiendo buscar, reportar y registrar votos utilizando diferentes estructuras de datos. Los datos necesarios para realizar estas acciones se encuentran en un archivo CSV, que funciona como base de datos, y cualquier voto registrado se almacenará en este archivo.
